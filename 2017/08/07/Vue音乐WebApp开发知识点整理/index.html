搜索节流

用一个延迟函数代理方法的执行。
原理：执行前检查是否存在定时器，有则清空并重新设置

export function debounce(func, delay) {
  let timer

  return function (...args) {
    if (timer) {
      clearTimeout(timer)
    }
    timer = setTimeout(() => {
      func.apply(this, args)
    }, delay)
  }
}
